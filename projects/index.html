    • TechRace 2K16
    (Android)

    Official companion app for the S.P.I.T. TechRace event. TechRace is a city-wide event where participants must visit various locations throughout the city. Locations are revealed by solving various riddles shown on the app.

    Features:

  • Barcode scanner to scan the code at each location
  • Details about visited locations are shown in app, along with the clues that have been solved already
  • Hint minigame: match the tiles within the move limit to unlock up to 2 clues throughout the event
